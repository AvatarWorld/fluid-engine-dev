Detailed Description

3-D triangle geometry.

This class represents 3-D triangle geometry which extends Surface3 by overriding surface-related queries.

Returns the closest surface normal from the given point otherPoint.

This function returns the "actual" closest surface normal from the given point otherPoint, meaning that the return value is not flipped regardless how Surface3::isNormalFlipped is set.
